# BioMonitor 2026

Projeto Front-End para o Concurso Agrinho 2026, subcategoria Programação Front-End.
Feito por: Geovana Valentin Carneiro, de Cidade Gaúcha - Paraná,
Cólegio Estadual Marechal Costa e Silva.

## Objetivo

Apresentar, de forma interativa, como tecnologia, sustentabilidade e produção rural podem atuar juntas para reduzir desperdícios, proteger biomas e fortalecer o agro com responsabilidade ambiental.

## Tecnologias Utilizadas

- HTML5 semântico
- CSS3 próprio, com responsividade por media queries
- JavaScript puro para interação com DOM, eventos, `localStorage`, modais, mapas interativos e animação de indicadores

## Funcionalidades

- Monitor interativo de biomas brasileiros
- Modal de pilares do ecossistema do agro
- Mapa simbólico para marcar localização do usuário
- Recorte regional do Paraná com regiões clicáveis
- Tema claro/escuro com preferência salva no navegador
- Layout responsivo com menu mobile
- Botão de voltar ao topo

## Organização

- `index.html`: estrutura do site
- `style.css`: estilos visuais e responsividade
- `script.js`: regras de interação e atualização dinâmica
- `img/`: imagens utilizadas no projeto
- `som/`: sons utilizadas no projeto
